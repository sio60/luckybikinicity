## JujuPick – AI 사주 & 운세 앱

복잡한 회원가입 없이, 실제 상담처럼 대화하며 운세를 볼 수 있는 모바일 앱 프로토타입입니다.  
React Native로 핵심 UI를 구현하고, 향후 AI 서버 연동을 위해 질문/응답 JSON 스키마와 프롬프트 구조를 설계했습니다.

### 핵심 기능

- **회원가입 없이 즉시 사용** : 앱 실행 직후 바로 운세/사주 상담 시작
- **챗봇형 UI** : 실제 상담을 받는 듯한 채팅 인터페이스 구성
- **AI 기반 운세 분석 구조 설계**  
  - 이름, 생년월일, 성별 등 기본 정보 입력 플로우
  - 향후 AI 서버와 연동 가능한 요청/응답 JSON 스키마 정의
- **카테고리별 질문** : 연애운, 금전운, 건강운, 일·학업 운 등 원하는 영역만 골라 질문 가능

### 기술 스택

- **Frontend / App** : React Native, JavaScript  
- **Backend / DB(설계)** : Supabase(PostgreSQL, Auth)  
- **Infra** : GitHub (로컬 디버그 빌드 기반 프로토타입)
